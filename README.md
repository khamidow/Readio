# 🎧 Readio

### Modern Audiobook & PDF Reading Platform built with Flutter

![Flutter](https://img.shields.io/badge/Flutter-3.x-02569B?style=for-the-badge\&logo=flutter\&logoColor=white)

Readio is a modern digital reading platform that allows users to access PDF books and audiobooks from a centralized cloud library. Built with Flutter and powered by Supabase, it delivers a seamless reading and listening experience across Android and iOS devices.

## ✨ Features

### 📚 Digital Library

* Browse a collection of books
* Discover newly added content
* Search and explore available titles
* Organized and scalable content management

### 📄 PDF Reading

* Open and read PDF books
* Smooth page navigation
* Fast document loading
* Optimized reading experience

### 🎧 Audiobook Playback

* Stream audiobooks directly from the cloud
* Play, pause, seek, and resume playback
* Continue listening from the last position
* Dedicated audiobook player interface

### ☁️ Cloud-Powered Content

* Books stored and managed with Supabase
* Centralized content updates
* Real-time content availability
* Secure backend integration

### 🎨 Modern User Experience

* Clean and intuitive interface
* Responsive layouts
* Smooth animations and transitions
* Material Design implementation

### 📱 Cross-Platform Support

* Android application
* iOS application
* Single Flutter codebase

---

## 🚀 Core Functionality

### PDF Books

Users can access and read PDF books directly within the application without requiring external readers.

### Audiobooks

Users can stream and listen to audiobook versions of available titles through a dedicated audio player.

### Cloud Library

All content is managed through Supabase, enabling easy updates and centralized book management.

### Seamless Experience

A unified platform for reading and listening from a single library.

## 🚀 Getting Started

### Clone Repository

```bash
git clone https://github.com/khamidow/Readio.git
```

### Navigate to Project

```bash
cd Readio
```

### Install Dependencies

```bash
flutter pub get
```

### Run Application

```bash
flutter run
```
